2. 사용 기술

## 특징<br/>
- 냉장고를 등록 할 수 있는 것이 강점<br/>
- 회원 가입을 해야지만 냉장고를 등록 할 수 있음<br/>
- 한 아이디 당 하나의 냉장고를 가질 수 있음<br/>
- 재료들을 선택하여 등록하기 버튼을 누르면 냉장고에 등록<br/>
- 냉장고에서 재료를 선택 후 검색을 하면 선택한 재료가 포함된 레시피들이 전부 나옴<br/>
